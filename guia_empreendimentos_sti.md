## 🛠️ Notas para o Desenvolvedor

- **Uso de Imagens:** Priorizar as imagens de alta resolução (`1024x...`) para o fundo dos cards ou modais de detalhes.
- **Ícones:** Use ícones específicos para cada feature (ex: `wind` para ar condicionado, `shield` para alarme, `car` para vagas).
- **Foco em Leads:** Destacar no texto do Malibu a "negociação direta" e "entrada facilitada", pois são gatilhos de venda reais da empresa.
